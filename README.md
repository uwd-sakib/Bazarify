## ЁЯЫа Tech Stack

### Frontend
- React 18 + Vite
- Tailwind CSS
- React Router v6
- Axios
- Recharts

### Backend
- Node.js + Express
- MongoDB + Mongoose
- JWT Authentication
- Express Validator

## ЁЯУж Installation

### Prerequisites
- Node.js 18+ 
- MongoDB 6+
- npm or yarn

### Backend Setup

\`\`\`bash
cd backend
npm install
cp .env.example .env
# Update .env with your configuration
npm run dev
\`\`\`

### Frontend Setup

\`\`\`bash
cd frontend
npm install
npm run dev
\`\`\`

## ЁЯЪА Deployment

### Backend
- Compatible with Railway, Render, or any Node.js hosting
- Set environment variables in hosting platform
- Ensure MongoDB connection string is configured

### Frontend
- Build: `npm run build`
- Deploy to Vercel, Netlify, or any static hosting
- Set API base URL in environment variables

## ЁЯУЭ Default Admin Credentials
